# Mac Utility UX

Menu bar utilities win on speed, clarity, and low friction.

## Rules

- Prefer instant actions over heavy confirmation flows.
- Do not steal focus unnecessarily.
- Preserve context when switching modes.
- Use tooltips for icon-only controls.
- Make errors actionable, not generic.
- Remember user preferences when that reduces repeated setup.

If a new workflow feels noisy, slow, or confusing, simplify it before shipping.
